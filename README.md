# Gradio Groq-Based Simple Chatbot

A minimal chatbot built using **Gradio** and **Groq LLM APIs**, designed for experimenting with prompt-based conversational AI through a simple web interface.

This project focuses on clarity and simplicity rather than production-scale complexity.

---

## Features

- Interactive chatbot UI using Gradio  
- Uses Groq-hosted LLMs for fast inference  
- Prompt-driven conversation flow  
- Simple Python structure, easy to extend  
- Local `.env` support for API key management
